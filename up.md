## Phase 5: The "Discovery" Overhaul (Visual & Interactive)

## Goal
Transform the dull dashboard into a high-end, image-rich news platform inspired by Inshorts and Medium.

## Tasks
1. **Image Integration:** Update the Scraper to fetch the `originalimage` or `thumbnail` URL from the Wikipedia API for every article.
2. **Category Intelligence:** Use an LLM agent to categorize each viral post into tags: #Geopolitics, #Entertainment, #Science, #WorldNews.
3. **The "Hero" UI:** - Add a "Top Story" section with a large, high-resolution background image and a bold headline.
   - Change the grid to "Magazine Style"—alternating card sizes for a more dynamic feel.
4. **Interactive Exploration:** - Add a "Daily Briefing" sidebar that summarizes the top 3 global events of the day.
   - Implement a "Swipe/Scroll" mode similar to Inshorts for mobile users.
5. **Aesthetics:** Use a deep charcoal background with "Glassmorphism" (blurred transparency) for the cards.
